# FlightInfoApi

FlightInfoApi is an ASP.NET Core Web API that provides flight information using the AeroDataBox API.  
It supports mock data for testing and includes usage tracking with throttling.

## Features
- Real-time flight info via AeroDataBox
- Mock mode for development/testing
- Usage tracking and API throttling
- Swagger/OpenAPI documentation

## Getting Started
1. Clone the repository
2. Configure `appsettings.Development.json` with your connection string and API key
3. Run the project
